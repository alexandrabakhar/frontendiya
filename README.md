# Поиск пользователей GitHub

## О проекте

"Поиск пользователей GitHub" - это веб-приложение, которое позволяет искать профили пользователей GitHub и просматривать подробную информацию о них.

## Основные функции

- **Поиск пользователей GitHub**: Введите имя пользователя, чтобы найти его профиль на GitHub.

## Особенности проекта:

- Используется архитектура Feature-Sliced Design, ориентированная на потребности бизнеса и гарантирующая чистоту разработки за счет изоляции модулей.

## Начало работы:

- [FSD v2.0.0](https://feature-sliced.design/ru/)
- [pnpm v8.9.2](https://pnpm.io/)
- [Node.js v18.12.1](https://nodejs.org/en)

Для того, чтобы клонировать проект к себе, выполните команду:

```sh
$ git clone https://github.com/alexandrabakhar/frontendiya
```

Для установки зависимостей, выполните команду:

```sh
$ pnpm i
```

Чтобы запустить сервер для разработки, выполните команду:

```sh
$ pnpm dev
```
